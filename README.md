# my-git-client

Git 작업을 수행할 수 있는 심플한 GUI 클라이언트입니다.

## 작업 기간

- **2024.01.19** ~ **2024.03.01**

## 기술 스택 및 개발 환경

### 주요 기술

- **.NET** - v8.0.101  
- **Avalonia** - v11.0.7  
- **LibGit2Sharp** - v0.29.0  

### 개발 환경

- Windows에서 테스트 및 구동 확인

## 설치 및 실행

아래 단계를 따라 프로젝트를 설치 및 실행할 수 있습니다.

### 1. 프로젝트 빌드

```bash
dotnet build
```

### 2. 애플리케이션 실행

```bash
dotnet run
```

## 기능 목록

현재 구현된 기능 목록입니다:

- 로컬 Git 저장소 연결
- 최신 브랜치 조회
- 커밋, 풀, 푸시, 리셋 작업 수행
- 커밋 히스토리 확인
- 변경된 파일 확인
